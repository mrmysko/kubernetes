# Cert-Manager

Deploys some things used by [Cert-Manager](https://github.com/cert-manager/cert-manager)

This chart does <b>NOT</b> deploy cert-manager

## Prerequisites

- [Cert-Manager](https://github.com/cert-manager/cert-manager)
- Helm >= 3.10
